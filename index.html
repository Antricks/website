About

This website is supposed to become a programming related blog-ish thing where I share my progress on different projects or some CTF writeups in case I feel like making some. I'm not very actively working on this website since I'm currently lacking time to do so. The code of this website is, just like most of the things I do, completely open source. I'm a big advocate of both open source and privacy in general.

You may want to check out https://demo.antricks.dev/ for some web-based things I already made.
